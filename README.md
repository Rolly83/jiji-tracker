# 📈 Jiji Price & Demand Market Tracker

A premium, cloud-deployed e-commerce analytics dashboard designed to monitor asset valuations, calculate live pricing distributions, and track market dynamics across the Nigerian secondary marketplace.

## 🚀 Live Application
Try the interactive web tool here: **[Jiji Market Tracker](https://jiji-market-tracker.onrender.com)**

## ✨ Core Features
* **Anti-Block Data Pipeline:** Dynamically intercepts marketplace metrics using a high-performance cross-origin network proxy architecture.
* **Smart Valuation Matrix:** Instantly evaluates product classes (Automotive, Mobile Electronics, etc.) and routes them through a multi-million Naira baseline algorithm.
* **Interactive Data Visualization:** Maps market spread density via responsive dark-themed box-plot analytics.
* **Supabase Cloud Synchronization:** Securely captures real-time price tracker triggers and client leads directly to an enterprise database table.

## 🛠️ Built With
* **Streamlit** - For the high-performance dark corporate user interface.
* **Plotly Express** - For multi-variable price spread graphing.
* **Supabase** - Postgres cloud storage structure.
* **Render** - Secure web application hosting infrastructure.
